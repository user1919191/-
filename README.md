本项目基于Java开发,结合Redis,MQ,ES实现功能扩展和性能提升。在项目中实现了题目,题库的增删改查和分页查询。管理员可以增加修改删除题目题库关联。并且通过逻辑删除对数据留底。用户可以设置自己的用户名片和进行发帖操作,并且用户可以点赞和收藏帖子。以下为项目架构图:
